## Version 1 System Flow

1. Upload CV
2. Upload Job Description
3. Extract text from both documents
4. Extract technical skills using a predefined skill dictionary
5. Compare CV skills against JD skills
6. Calculate match score
7. Display:
   - Match score
   - Matched skills
   - Missing skills
   - Basic recommendations

## Version 1 Scoring Logic

Match Score = (Number of matched JD skills / Total JD skills) × 100
